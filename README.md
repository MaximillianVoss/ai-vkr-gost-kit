# AI VKR / DP GOST Kit

Самодостаточный комплект для ИИ-агента, который должен готовить и оформлять ВКР, дипломный проект или пояснительную записку по пользовательскому шаблону Word и ГОСТ с явным выбором профильной методички.

Состав:
- `word_agent_toolkit.py` - автономный CLI для работы с `.dotm/.docx`: создание документа из шаблона, применение ГОСТ-профиля, обновление оглавления, экспорт PDF, добавление приложения с листингом.
- `ai-client-vkr-gost-spec.json` - машиночитаемый профиль с маршрутами выбора между дипломным проектом СПО и ВКР магистратуры.
- `Профиль для ИИ агента - ВКР и ДП по ГОСТ.docx` - редактируемый Word-образец, который можно открыть, поправить и затем передать другому ИИ.
- `Профиль для ИИ агента - ВКР и ДП по ГОСТ.pdf` - PDF-предпросмотр того же образца.

Что требуется извне:
- локальный шаблон Word `.dotm`, который не хранится в репозитории;
- при необходимости локальный PDF с методическими указаниями;
- Windows и установленный Microsoft Word для операций COM-автоматизации;
- Python с пакетами `python-docx` и `pywin32`.

Установка зависимостей:

```powershell
python -m pip install -r requirements.txt
```

Важно:
- пути к шаблону и методичке в JSON-спецификации являются машинно-зависимыми и должны быть заменены при переносе комплекта на другой компьютер;
- временные файлы рендера и проверки рекомендуется складывать в `tmp/docs/`;
- итоговый Word/PDF-профиль агента в этом комплекте считается каноническим;
- если пользователь написал только `ВКР`, агент не должен автоматически выбирать магистерский профиль: сначала нужно понять, это СПО/дипломный проект или ВО/магистратура.

Как выбирать методичку:

- `МУ к ДП.pdf`: использовать для дипломного проекта или ВКР СПО, когда работа относится к колледжу/техникуму и имеет четырехглавую прикладную структуру с обязательной экономической главой.
- `МУ ВКР.pdf`: использовать для ВКР магистратуры или другой ВКР высшего образования, когда обязательны реферат, анализ актуальности, формальное проектирование, технико-экономическое обоснование, анализ рисков и комплект сопроводительных документов.
- Если пользователь дал обе методички или сказал только `ВКР`, сначала определить уровень образования и только потом выбирать профиль оформления.

Быстрый старт:

```powershell
python word_agent_toolkit.py resolve-profile --work-type "дипломный проект" --education-level "СПО"
python word_agent_toolkit.py resolve-profile --work-type "магистерская ВКР" --education-level "магистратура"
python word_agent_toolkit.py inspect-template --template "C:\path\to\your-template.dotm"
python word_agent_toolkit.py generate-agent-brief --template "C:\path\to\your-template.dotm" --output "Профиль для ИИ агента - ВКР и ДП по ГОСТ.docx"
python word_agent_toolkit.py export-pdf --input "Профиль для ИИ агента - ВКР и ДП по ГОСТ.docx" --output "Профиль для ИИ агента - ВКР и ДП по ГОСТ.pdf"
```
