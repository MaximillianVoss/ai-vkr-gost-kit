# AI VKR GOST Kit

Самодостаточный комплект для ИИ-агента, который должен готовить и оформлять ВКР, курсовую работу, дипломный проект или пояснительную записку по пользовательскому шаблону Word и ГОСТ.

Состав:
- `word_agent_toolkit.py` - автономный CLI для работы с `.dotm/.docx`: создание документа из шаблона, применение ГОСТ-профиля, обновление оглавления, экспорт PDF, добавление приложения с листингом.
- `ai-client-vkr-gost-spec.json` - универсальный профиль правил оформления и поведения ИИ-агента.
- `Профиль для ИИ агента - ВКР по ГОСТ.docx` - редактируемый Word-образец, который можно открыть, поправить и затем передать другому ИИ.
- `Профиль для ИИ агента - ВКР по ГОСТ.pdf` - PDF-предпросмотр того же образца.

Что требуется извне:
- локальный шаблон Word `.dotm`, который не хранится в репозитории;
- при необходимости локальный PDF с методическими указаниями;
- Windows и установленный Microsoft Word для операций COM-автоматизации;
- Python с пакетами `python-docx` и `pywin32`.

Установка зависимостей:

```powershell
python -m pip install -r requirements.txt
```

Важно:
- пути к шаблону и методичке в JSON-спецификации являются машинно-зависимыми и должны быть заменены при переносе комплекта на другой компьютер;
- временные файлы рендера и проверки рекомендуется складывать в `tmp/docs/`;
- итоговый Word/PDF-профиль агента в этом комплекте считается каноническим.

Быстрый старт:

```powershell
python word_agent_toolkit.py inspect-template --template "C:\path\to\your-template.dotm"
python word_agent_toolkit.py generate-agent-brief --template "C:\path\to\your-template.dotm" --output "Профиль для ИИ агента - ВКР по ГОСТ.docx"
python word_agent_toolkit.py export-pdf --input "Профиль для ИИ агента - ВКР по ГОСТ.docx" --output "Профиль для ИИ агента - ВКР по ГОСТ.pdf"
```
